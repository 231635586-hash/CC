📋 v0.2.0-M2 到货处理流程

• 完整链路：派车 → dispatched → entering → loading → leaving → in_transit → arrived_by_gps → driver_confirmed → customer_signed → completed

• 司机端：到达客户园区后，GPS 自动判定 → H5 显示「确认到达」按钮

• PC 端：司机确认后，PC 生成客户签收链接（带 24h token）→ 调度员发给客户

• 客户签收：客户扫码进入 → 上传 ≥1 张照片 → 确认签收 → 订单完成
