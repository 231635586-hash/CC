📱 华翔司机端 H5 — PhoneFrame 预览

iPhone 14 · 390 × 844 · 5 Tab 单页架构

1 个主页面 + 1 个详情页
主页面（5 Tab 集成） 工作台 / 派车单 / 消息 / GPS / 我的
详情页（点击跳转） 点击列表中某条订单跳转
支持返回保留列表状态

📋 架构说明

• 单页 5 Tab：减少跳转 = 减少司机操作摩擦。所有功能一屏可达。

• 详情跳转：点击列表条目 → 跳详情页 → 返回时 Tab 不动、滚动位置保留（uni-app 原生行为）。

• 设计 Token：CSS Variables 全局生效，1 处改全局同步。
