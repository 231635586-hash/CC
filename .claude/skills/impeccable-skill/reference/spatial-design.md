# Spatial Design（空间设计）

布局和视觉层级的精确性和有意性。

---

## 间距刻度

使用 4pt 基准（不是 8pt）以获得精细度而不出现尴尬的中间值：

`4, 8, 12, 16, 24, 32, 48, 64, 96px`

使用 `gap` 而不是 margins 以消除折叠问题。

---

## 视觉层级：分层多个维度

绝不依赖单一维度进行层级。结合：
- **大小**（级别之间最小 3:1 比例）
- **字体排版字重**（粗体 vs 常规）
- **颜色对比度**
- **定位**
- **留白**

**眯眼测试**：眯眼检查层级是否保持清晰——最重要的元素即使模糊也应该突出。

---

## 自适应网格

```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
}
```

创建基于可用空间自然适应的布局，无需媒体查询。

---

## 容器查询

允许组件响应其自身容器宽度而不是视口大小。相同组件可以在侧边栏中紧凑，在主内容区域中展开：

```css
@container (min-width: 400px) {
  .card { flex-direction: row; }
}
```

---

## 两种设计上下文

**品牌设计**：不对称、流体、对比度驱动。慷慨留白（100–200px 间隙），戏剧性比例跳跃。

**产品设计**：可预测、一致、结构化。系统间距刻度、清晰网格对齐。

---

## 触摸目标扩展

小视觉元素（24px 图标）需要至少 44px 的隐藏扩展触摸目标：

```css
.icon::after {
  content: '';
  position: absolute;
  inset: -10px;
}
```

---

## 语义命名

按角色命名间距 tokens（`--space-component-gap`），而不是按值（`--space-16`）。名称存活于设计变化；值变得误导。

**核心要点**：深思熟虑的间距、语义命名和分层层级创造精致、功能性的设计。
