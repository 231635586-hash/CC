# Motion Design（动效设计）

UI 动画时间和缓动的实用标准。深思熟虑的timing和材料选择比动画数量更重要。

---

## Timing

| 时长 | 用途 |
|----------|-----|
| 100–150ms | 即时反馈（按钮按压、切换） |
| 200–300ms | 状态变化（菜单、工具提示） |
| 300–500ms | 布局移动（手风琴、模态框） |
| 500–800ms | 入场效果（页面加载） |

**退出动画比入场动画快。** 退出使用入场的约 75% 时长。

---

## 缓动曲线

避免通用的 `ease` 函数。使用特定曲线：
- **Ease-out** 用于入场元素（减速就位）
- **Ease-in** 用于离场元素（加速离开）
- **指数曲线**（Quart、Quint、Expo）用于模仿物理的自然运动

避免 bounce 和弹性效果——它们感觉过时。

推荐：`cubic-bezier(0.25, 1, 0.5, 1)` 用于平滑减速。

---

## Transform/Opacity 之外的动画材料

优质界面利用：
- **模糊 / backdrop-filter**：深度和分层
- **Clip paths**：揭示和隐藏
- **Masks**：定向擦拭
- **阴影动画**：海拔变化

仅在它们增强平滑度且不会对目标设备造成性能压力时使用。

---

## 无障碍

前庭功能障碍影响约 40 岁以上成年人的 35%。`prefers-reduced-motion` 是必需的，不是可选的：

```css
@media (prefers-reduced-motion: reduce) {
  * { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; }
}
```

---

## 感知性能

利用骨架屏和渐进内容加载，而不是仅依赖动画速度。目标是感觉快，而不仅仅是快。

---

## 核心原则

添加传达状态、提供反馈和澄清层级的动效。切掉纯粹为装饰而存在的动效。
