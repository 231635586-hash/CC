# MES生产工单管理模块

> 生产工单管理模块高保真原型，含工单列表、详情、新建/编辑、报工、拆分/合并等完整交互。

## 页面清单

| 页面 | 文件 | 说明 |
|------|------|------|
| 工单列表页 | `index.html` 内嵌 | 搜索/筛选/排序/分页 |
| 工单详情页 | `index.html` 内嵌 | 信息展示、状态流转、工序进度、设备状态 |
| 工序报工页 | `index.html` 内嵌 | 扫码报到、良品/不良品录入 |
| 新建工单弹窗 | `index.html` 内嵌 | 表单填写、产品选择 |
| 工单拆分弹窗 | `index.html` 内嵌 | 按比例拆分计划数量 |
| 工单合并弹窗 | `index.html` 内嵌 | 多选合并 |

## 功能覆盖

### 核心功能
- ✅ 工单列表（搜索、筛选车间/状态、排序、分页）
- ✅ 工单详情（信息展示、操作按钮、工序进度）
- ✅ 新建工单（产品选择、数量、交期、优先级）
- ✅ 状态流转（创建→审核→已下达→进行中→已完成）
- ✅ 工序级报工（良品/不良品/工时报工）
- ✅ 工单拆分（按子工单数量等比拆分）
- ✅ 工单合并（多选合并）

### 边界场景
- ✅ 缺料预警展示（在详情页）
- ✅ 工单暂停/恢复（由流程处理人操作）
- ✅ 报工时不良品触发质检提示
- ✅ 完工后自动触发入库单

### 跨模块关联
- ✅ 质检模块（报完工触发）
- ✅ 库存模块（完工触发入库单）
- ✅ 设备模块（产线设备状态展示）

## 技术栈

- **纯前端**：HTML + CSS + JavaScript，无框架依赖
- **无需构建**：直接在浏览器打开 `index.html` 即可预览

## 运行方式

```bash
# 直接在浏览器打开
open mes-workorder/index.html

# 或使用本地服务器
npx serve mes-workorder
```
